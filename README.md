A Chat App built using :
- Typescript 
- MERN stack
- Socket io
- Tailwind And Shadcn 
- Multer
- Cloundinary


Features : 
- Real time Chats
- send and receive Emoji's
- Typing Indicator
- Image sending
- Online status 
- Make Group Chats
- Add / remove people from Groups if you're admin
